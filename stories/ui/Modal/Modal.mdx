# Modal Components

`Modal` 컴포넌트는 다양한 모달 기능을 제공합니다. 아래는 `LoginModal`, `SignUpModal`, `EditProfileModal`과 같은 예제입니다.

## Props

`Modal` 컴포넌트는 두 가지 주요 props를 받습니다:

- `openModal`  
  이 prop은 모달이 열려 있는지 여부를 제어하는 **boolean** 값입니다. `true`로 설정하면 모달이 열리고, `false`로 설정하면 닫힙니다.

- `handleModalClose`  
  모달을 닫는 동작을 수행하는 **함수**입니다. 이 함수를 호출하면 모달이 닫히는 로직이 실행됩니다.

각 props는 모달의 동작과 상태 관리를 위한 핵심 역할을 합니다.
